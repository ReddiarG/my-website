## Overview
As a personal learning initiative, I built a full-stack web-based chat application powered by a Retrieval-Augmented Generation (RAG) pipeline. The goal was to understand how Large Language Models (LLMs) can be enhanced with external, structured knowledge sources â€” going beyond theory into a hands-on system implementation.

## This project gave me deep exposure to:

- **Embedding Models**
- **Chunking Strategies**
- **Vector Similarity Search**
- **LLM Prompt Engineering**
- **FastAPI and WebSockets based Backend Integration**
- **Building full-stack Generative AI application from scratch**
